```
・获取 Account ID
・获取 API Token  选择所有账户，所有区域

2. Fork repository　运行github workflower， 自動部署 Cloudflare Worker（Worker默认名字为 gemini ） 

3. Worker绑定域名（例如：gemini.grapehut.us.kg） 　※然后就可以调用此域名了
```
